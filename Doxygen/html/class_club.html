Detailed Description

Class that represents the whole club.

Parameters
minisPointer to object of class Level :: Minis
juvenilesPointer to object of class Level :: Juveniles
juniorsPointer to object of class Level :: Juniors
seniorsPointer to object of class Level :: Seniors
leaveprogramBool that tells when to leave program
singleton_instancePointer to only object of class Club
filename_playersName of the players database text file
filename_trainingsName of the trainings database text file
filename_tournamentsName of the tournaments database text file

This class keeps access through pointers to all of the levels of the club. Used mostly for I/O operations on text files and provide access to all players or all levels.
