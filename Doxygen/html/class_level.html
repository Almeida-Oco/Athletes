Detailed Description

Class used to represent a level of the Club.

class Level

Parameters
playersAll of the players of the level
eventsAll of the events of the level
coachName of the coach of the level

There are only 4 levels in the Club, Minis Juniors Juveniles and Seniors which all inherit from this class.
